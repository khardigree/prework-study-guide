HTML

  • HTML is responsible for the "what" that is in a website. The words, images, and layout are all HTML.
  • The head element contains info about the webpage. This is invisible to the user.
  • The meta element contains info about the webpage, and is what is housed in the head element listed above. This is also invisible to the user.
  • The body element conatins the information that is presented to the user. This is the "website" that the user interacts with.

Git

  • Git is a version control system that allows developers to store their code, work from different computers, work in tandem with other developers without overlap, track versions, projects, etc.
  • GitHub is a webpage that almost acts as a social media platform for sharing and viewing projects, as well as a repository system for coding projects, provides a workflow system, etc for developers
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches you to the main branch
  • git add -A: stage your changes
  • git commit -m "note regarding changes": commits files to working branch
